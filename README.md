
# API para tienda (Práctica)

API desarrollada en express con Typescript como ejemplo para clase de fundamento de base de datos.

Descargar módulos
* npm install

Iniciar proyecto
* npm run dev

Construir app
* npm run start

# Ejemplo de variables de entorno (.env)
DB_HOST=localhost

DB_PORT=3306

DB_USER=pedro

DB_PASSWORD=1234

DB_NAME=tienda_familiar

PORT=8000
